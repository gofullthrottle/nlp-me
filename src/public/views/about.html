NLP ME

Welcome to NLP ME, a place where you can "chat" with natural language processing APIs.

What is natural language processing?

Natural language processing is a subfield of artifical intelligence that involves extracting meaning from text and spoken language. These days NLP is rapidly being adopted by companies to analyze all sorts of data from their customers. For instance, a company might use NLP to analyze tweets with their hashtag to see if people are saying good or bad things about their product.

What is an API?

An API, or an application programming interface, is a connection through which different pieces of software communicate. In the context of client-server web applications, the term often refers to a REST API, which is an interface that the client (your computer) can use to communicate with a server (a computer in the cloud which contains all of the application's data).

What is this website?

This is a web application that allows you to see what kind of data NLP APIs can extract from text. Use the chat box to see what any of the APIs listed below can figure out about what you type!
